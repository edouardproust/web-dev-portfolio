# EP Portfolio

Portfolio of Edouard Proust, web developer (PHP , Symfony and Javascript)

## Technologies & Requirements

- PHP 7.4
- Symfony 5
- Symfony CLI
- Composer
- Docker
- Docker-compose
- Nodejs & yarn

```bash
symfony check:requirements
```

## Launch

```bash
composer install
yarn install
yarn dev
docker compose up -d
symfony serve -d
```

## Usefull commands

To build Encore files automatically:
```bash
yarn watch
```
